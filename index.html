Live Pit Scout Sync

Lets your whole team scout different robots simultaneously — all data merges in real time. Uses a free Firebase Realtime Database.

One-time Setup (5 min)
  1. Go to console.firebase.google.com
  2. Add project → any name → Create project
  3. Left sidebar: Build → Realtime Database → Create Database
  4. Choose "Start in test mode" → Enable
  5. Copy the database URL (ends in .firebaseio.com)

The Blue Alliance API Key

2026 REBUILT · Scoring Reference

FUEL

5.91" foam ball into active HUB (41.7" hex at 72"). 1 pt per FUEL in active HUB. Inactive HUB = 0 during shifts.

TOWER Climb

L1: 15 auto / 10 teleop
L2: 20 pts teleop
L3: 30 pts teleop
Max 2 robots on tower.

Ranking Points

Win: 3 RP   Tie: 1 RP
ENERGIZED: 100+ FUEL
SUPERCHARGED: 360+ FUEL
TRAVERSAL: 50+ tower pts

HUB Shifts

4 × 25s shifts. Auto FUEL winner controls their HUB. Both HUBs active in auto (20s), transition (10s), endgame (30s).

Obstacles

BUMP: 6.5" tall — drive over.
TRENCH: 40.25" tall — drive under.

Climb Combos

L3+L3 = 60 pts ✓
L3+L2 = 50 pts ✓ (TRAVERSAL)
L3+L1 = 40 pts ✗
L2+L2 = 40 pts ✗
